# 🚀 5分钟快速部署指南

## 第一步：准备工作（10分钟）

### 1. 下载并解压项目
解压 `construction-ops-dashboard-ready.zip` 到你的电脑

### 2. 安装必要软件

#### Node.js（必需）
- 访问：https://nodejs.org/
- 下载并安装 LTS 版本
- 安装后打开命令行输入 `node --version` 验证

#### Git（必需）
- 访问：https://git-scm.com/
- 下载并安装
- 安装后打开命令行输入 `git --version` 验证

### 3. 注册账号

#### GitHub（必需）
- 访问：https://github.com/
- 点击 Sign up 注册

#### Vercel（必需）
- 访问：https://vercel.com/
- **重要**：选择 "Continue with GitHub" 使用 GitHub 账号登录

## 第二步：发布到 GitHub（3分钟）

### 配置 Git（第一次使用需要）
打开命令行/终端，输入：
```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

### 创建 GitHub 仓库
1. 访问：https://github.com/new
2. 填写：
   - Repository name: `construction-ops-dashboard`
   - 选择 Public（公开）
   - ⚠️ **不要勾选任何初始化选项**
3. 点击 "Create repository"
4. **复制仓库地址**（形如：https://github.com/xxx/construction-ops-dashboard.git）

### 推送代码

#### Windows 用户：
1. 在项目文件夹中双击 `deploy.bat`
2. 按提示粘贴你的仓库地址
3. 等待完成

#### Mac/Linux 用户：
1. 打开终端，进入项目目录
2. 运行：`./deploy.sh`
3. 按提示粘贴你的仓库地址
4. 等待完成

## 第三步：部署到 Vercel（2分钟）

1. 访问：https://vercel.com/
2. 点击 "New Project"
3. 找到并选择 `construction-ops-dashboard`
4. 点击 "Deploy"
5. 等待 1-2 分钟

## 完成！🎉

部署完成后，Vercel 会显示你的网站地址，类似：
```
https://construction-ops-dashboard.vercel.app
```

点击即可访问你的仪表盘！

---

## 🔄 如何更新网站

当你修改代码后：

```bash
git add .
git commit -m "更新说明"
git push
```

Vercel 会自动重新部署！

## ❓ 遇到问题？

### Git 推送要求输入密码？
1. 访问：https://github.com/settings/tokens
2. 生成 Personal Access Token
3. 勾选 `repo` 权限
4. 推送时用 token 作为密码

### Vercel 找不到仓库？
确保：
1. 使用 GitHub 账号登录 Vercel
2. 授权 Vercel 访问你的仓库

### 网站打不开或空白？
1. 等待 2-3 分钟让部署完成
2. 清除浏览器缓存
3. 查看 Vercel 部署日志
